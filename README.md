AI-Based Real-Time Holographic Communication System using 3D Human Pose Reconstruction
This project focuses on building a real-time intelligent system that captures live video through a webcam, detects human body movements, reconstructs them into a 3D pose, and visualizes the output in a holographic-style interface. It integrates computer vision, deep learning, and 3D rendering to simulate next-generation communication systems.
